# one-db-mitra

Aplikasi manajemen data karyawan dan organisasi berbasis ASP.NET Core MVC.

## Cara Menjalankan

1. Pastikan sudah terinstall .NET SDK (minimal versi yang sesuai dengan project).
2. Clone repository ini:
   ```
   git clone https://github.com/lulusbagos/one-db-mitra.git
   ```
3. Masuk ke folder project:
   ```
   cd one-db-mitra
   ```
4. Jalankan aplikasi:
   ```
   dotnet run
   ```

## Konfigurasi
- Ubah file `appsettings.json` sesuai kebutuhan database dan konfigurasi lainnya.
